    What is it

    Vector2js is an open-source project that provide simple 2D vectors for JavaScript applications.
    It works on all major browsers + Node.js with 100% test coverage.

    Install

    You can install Vector2js from npm: npm install vector2js

    If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually.
