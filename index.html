    What is it

    Vector2js is an open-source project that provide simple 2D vectors for JavaScript applications.
    It works on all major browsers + Node.js with 100% test coverage.

    Install

    You can install Vector2js from npm: npm install vector2js Or you can just download the repo and include the main JavaScript file from /dist/.

    Usage & API

    Vector2js exports one main object, Vector. To instanciate a new vector, simply use: // if in nodejs or using browserify, you need to require the vector class first:
    var Vector = require("node2js");

    // create a new vector
    var vec = new Vector(1,1);     Or, you can create a new vector from a given angle (its length will be 1 by default): // create vector with length of 1 from radian:
    var vec = Vector.from_radian(0.785398);

    // create vector with length of 1 from degree:
    var vec = Vector.from_degree(45);     Once you created a vector, you can use all the API below:
